## What is Auth-Agent?

Auth-Agent is a complete OAuth 2.1 / OpenID Connect authentication system designed specifically for AI agents. It provides secure, standards-compliant authentication with automatic token management, React components, and comprehensive SDKs.

## Architecture

```mermaid
sequenceDiagram
    participant User
    participant App
    participant SDK
    participant AuthServer
    participant Database

    User->>App: Click "Sign in with Auth-Agent"
    App->>SDK: redirectToLogin()
    SDK->>SDK: Generate PKCE challenge
    SDK->>AuthServer: GET /oauth2/authorize
    AuthServer->>User: Show auth page
    User->>AuthServer: Enter credentials
    AuthServer->>Database: Verify agent
    Database-->>AuthServer: Agent valid
    AuthServer->>App: Redirect with code
    App->>SDK: handleCallback()
    SDK->>AuthServer: POST /oauth2/token (with PKCE verifier)
    AuthServer-->>SDK: Access + Refresh tokens
    SDK->>App: Authentication complete
    App->>User: Show dashboard
```
